# Assets Directory

This directory contains assets for the Audio Separator project.

## Structure

- `models/` - Pre-trained model files (if any)
- `examples/` - Example audio files for testing
- `configs/` - Configuration files

## Model Files

Pre-trained models should be placed in the `models/` subdirectory. These are typically large files and may need to be downloaded separately.

## Example Audio

Example audio files for testing and demonstration can be placed here. Note that large audio files should be added to `.gitignore`.
