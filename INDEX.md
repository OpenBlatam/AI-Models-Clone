## 🚀 Quick Start Guide

### 1. Installation
```bash
# Clone repository
git clone https://github.com/truthgpt/truthgpt-spec.git
cd truthgpt-spec

# Install dependencies
pip install -r requirements.txt

# Run tests
pytest tests/
```

Start with the main entry points:
1. **README.md** - Overview of the entire specification
2. **docs/design-rationale.md** - Understanding the design philosophy
