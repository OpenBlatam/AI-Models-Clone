# Math Operations Package

A comprehensive collection of mathematical functions with multiple implementations and optimizations.

## Structure

```
math_operations/
├── __init__.py              # Package initialization
├── basic_operations.py      # Simple math functions
├── advanced_operations.py   # Classes and optimized implementations
├── utils.py                 # Helper functions
├── tests/                   # Test suite
│   └── test_operations.py
└── README.md               # This file
```

## Installation

```bash
# No external dependencies required for basic operations
```

## Usage

### Basic Operations

```python
from math_operations import add, multiply, divide, power

# Simple calculations
result = add(5, 3)        # 8
product = multiply(4, 7)  # 28
quotient = divide(10, 2)  # 5.0
exponent = power(2, 3)    # 8.0
```

### Advanced Operations

```python
from math_operations import PowerCalculator, MathCalculator

# Power calculations with different methods
calc = PowerCalculator()
result1 = calc.power_basic(2, 3)   # 8.0
result2 = calc.power_math(2, 3)    # 8.0
result3 = calc.power_numpy(2, 3)   # 8.0

# Comprehensive calculator
math_calc = MathCalculator()
sqrt_result = math_calc.sqrt(16)   # 4.0
log_result = math_calc.log(100, 10) # 2.0
```

### Utilities

```python
from math_operations import validate_input, round_result

# Input validation
is_valid = validate_input(5, int)  # True

# Result formatting
formatted = round_result(3.14159, 2)  # 3.14
```
