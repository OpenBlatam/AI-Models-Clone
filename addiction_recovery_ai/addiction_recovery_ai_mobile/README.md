# Addiction Recovery AI - Mobile App

Aplicación móvil React Native TypeScript Expo para el sistema de recuperación de adicciones con IA.

## 🚀 Características

- ✅ Autenticación completa (registro/login)
- ✅ Evaluación de adicciones
- ✅ Seguimiento de progreso en tiempo real
- ✅ Planes de recuperación personalizados
- ✅ Prevención de recaídas
- ✅ Coaching con IA
- ✅ Análisis y estadísticas
- ✅ Notificaciones inteligentes
- ✅ Gamificación y logros
- ✅ Chatbot de apoyo
- ✅ Y mucho más...

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Iniciar en desarrollo
npm start

# Ejecutar en Android
npm run android

# Ejecutar en iOS
npm run ios
```

## ⚙️ Configuración

Crea un archivo `.env` en la raíz del proyecto:

```env
API_BASE_URL=http://localhost:8018
API_PREFIX=/recovery
```

## 📱 Estructura del Proyecto

```
src/
├── components/     # Componentes UI reutilizables
├── screens/        # Pantallas de la aplicación
├── services/       # Servicios API
├── types/          # Tipos TypeScript
├── hooks/          # Custom hooks
├── store/          # Estado global (Zustand)
└── utils/          # Utilidades
```

## 🔗 API

La app se conecta a la API de Addiction Recovery AI en:
- Base URL: Configurable en `.env`
- Endpoints: Ver documentación de la API
