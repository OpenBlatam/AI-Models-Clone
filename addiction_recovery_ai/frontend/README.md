# Frontend - Addiction Recovery AI

## 🚀 Descripción

Frontend moderno para el sistema de Addiction Recovery AI construido con Next.js y React.

## ✨ Características

- 📱 Diseño responsive y moderno
- 📊 Visualizaciones interactivas con Chart.js
- 💬 Chatbot integrado
- 📈 Dashboard en tiempo real
- 🔔 Notificaciones push
- 📱 PWA (Progressive Web App)

## 🛠️ Instalación

```bash
cd frontend
npm install
```

## 🚀 Desarrollo

```bash
npm run dev
```

El frontend estará disponible en `http://localhost:3000`

## 📦 Build para Producción

```bash
npm run build
npm start
```

## 🔧 Configuración

Configurar la URL de la API en `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8018
```

## 📁 Estructura

```
frontend/
├── pages/          # Páginas de Next.js
├── components/     # Componentes React
├── hooks/          # Custom hooks
├── utils/          # Utilidades
├── styles/         # Estilos
└── public/         # Archivos estáticos
```
