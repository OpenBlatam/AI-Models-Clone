## 📋 Overview

This directory (`agents/backend/onyx/server/features`) contains the modular feature implementations of the Onyx Server Architecture. Each subdirectory represents a distinct capability or independent agent system.

## 🗂 Feature Map

### Core Capabilities
- [**Business Agents**](./business_agents/README.md): Orchestration layer for enterprise workflows.
- [**Workflow Orchestrator**](./workflow_orchestrator_ai/README.md): Event-driven DAG execution engine.
- [**Universal Benchmark**](./universal_model_benchmark_ai/README.md): Evaluation suite for LLMs.

### Specialized Agents
- [**CyberSec Toolkit**](./cybersec_toolkit/README.md): Security auditing and scanning.
- [**Copywriting System**](./copywriting_system/README.md): Marketing content generation.
- [**Video Editor AI**](./video_editor_ai/README.md): Automated video post-production.
- [**Virtual Assistant**](./virtual_assistant/README.md): Personal productivity agent.

### Vertical Solutions
- [**Virtual Event Planner**](./virtual_event_planner/README.md): Conference logistics management.
- [**Virtual Fashion Stylist**](./virtual_fashion_stylist_ai/README.md): E-commerce recommendation engine.
- [**Proposal Generator**](./proposal_generator/README.md): Sales document automation.

## 🤝 Contributing

To add a new feature:

1. Create a new directory.
2. Implement your logic.
3. Add a `README.md` following the [Enterprise Standard](../../../README.md#documentation-standards).
4. Link it here and in the main README.
