# AI Detector Multimodal

> Part of the [Blatam Academy Integrated Platform](../README.md)

AI-generated content detector with multimodal forensic analysis.

## Features

- ✅ AI-generated text detection
- ✅ AI-generated image detection
- ✅ AI-generated audio detection
- ✅ AI-generated video detection
- ✅ Forensic prompt analysis
- ✅ AI model identification (GPT, Claude, Gemini, LLaMA, etc.)
- ✅ AI-generated content percentage
- ✅ Generation parameter analysis

## Endpoints

### POST `/ai-detector/detect`
Detects if content was generated by AI.

**Request:**
```json
{
  "content": "Text to analyze...",
  "content_type": "text",
  "metadata": {}
}
```

**Response:**
```json
{
  "is_ai_generated": true,
  "ai_percentage": 85.5,
  "detected_models": [
    {
      "model_name": "gpt-4",
      "confidence": 0.85,
      "provider": "OpenAI"
    }
  ],
  "primary_model": {
    "model_name": "gpt-4",
    "confidence": 0.85,
    "provider": "OpenAI"
  },
  "forensic_analysis": {
    "estimated_prompt": "Explain or describe: ...",
    "prompt_confidence": 0.6,
    "prompt_patterns": ["explanatory"],
    "generation_parameters": {
      "estimated_max_tokens": 500,
      "temperature": 0.7
    }
  },
  "confidence_score": 0.85,
  "detection_methods": ["pattern_matching", "statistical_analysis"],
  "processing_time": 0.15,
  "timestamp": 1234567890.123,
  "quality_info": {
    "writing_quality": 0.85,
    "paraphrase_likelihood": 0.15,
    "risk_score": 0.82,
    "reliability": "high"
  },
  "from_cache": false
}
```

### POST `/ai-detector/batch`
Batch detection for multiple contents.

### GET `/ai-detector/health`
Service health check.

### GET `/ai-detector/models`
Lists AI models the detector can identify.

### POST `/ai-detector/cache/clear`
Clears detection cache.

### GET `/ai-detector/cache/stats`
Gets cache statistics.

### GET `/ai-detector/stats`
Gets general detector statistics.

### GET `/ai-detector/history?limit=50`
Gets recent detection history.

### GET `/ai-detector/statistics`
Gets detailed detection statistics (detection rate, most common model, etc.).

## Alert System

The detector now includes an automatic alert system that notifies about:

- **High AI Confidence**: When percentage exceeds threshold (default 80%)
- **Specific Model Detected**: When a model is detected with high confidence (>80%)
- **Multiple Models Detected**: Indicates possible paraphrased content
- **Very High Confidence**: When confidence exceeds 90%
- **Inconsistent Detection**: When there are discrepancies between percentage and confidence

### Estimated Generation Parameters
- `max_tokens`: Based on text length
- `temperature`: Estimated based on variability (burstiness)
- `top_p`: Common estimation
- `presence_penalty` and `frequency_penalty`: Detected

### Forensic Evidence
- Text length (short/medium/long)
- Structure type
- Sentence structure (uniform/varied)
- Confidence based on detected models
- Estimated prompt instructions
