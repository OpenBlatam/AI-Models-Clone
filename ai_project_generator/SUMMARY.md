```
1. Autenticación (JWT/API Key)
   ↓
2. Rate Limiting Check
   ↓
3. Cache Check
   ↓
4. Generar Proyecto
   ├── Backend + Frontend
   ├── Tests + CI/CD
   ├── Validación
   └── Metadata
   ↓
5. Registrar Métricas
   ↓
6. Guardar en Cache
   ↓
7. Disparar Webhook
   ↓
8. Retornar Resultado
   ↓
9. Actualizar Dashboard (si está activo)
```
