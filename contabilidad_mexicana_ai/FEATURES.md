# Características - Contabilidad Mexicana AI

## 🎯 Funcionalidades Principales

### 1. Cálculo de Impuestos

Calcula impuestos para diferentes regímenes fiscales mexicanos:

- **ISR (Impuesto Sobre la Renta)**
  - RESICO con tramos progresivos
  - PFAE (Personas Físicas con Actividades Empresariales)
  - Sueldos y Salarios
  - Personas Físicas y Morales

- **IVA (Impuesto al Valor Agregado)**
  - Tasa general (16%)
  - Tasa cero
  - Exentos

- **IEPS (Impuesto Especial sobre Producción y Servicios)**
  - Cálculo según productos/servicios

**Características:**
- Desglose detallado por tramos
- Fórmulas paso a paso
- Fechas de pago y obligaciones
- Recomendaciones personalizadas

### 2. Asesoría Fiscal

Proporciona asesoría personalizada sobre temas fiscales:

- Elección de régimen fiscal
- Deducciones aplicables
- Optimización fiscal legal
- Estrategias de cumplimiento
- Análisis de situaciones específicas

### 3. Guías Fiscales

Genera guías completas sobre temas fiscales:

- Deducciones para emprendedores
- Facturación electrónica CFDI 4.0
- Trámites del SAT
- Declaraciones fiscales
- Regularización fiscal

**Niveles de detalle:**
- Básico: Información esencial
- Intermedio: Con ejemplos
- Completo: Guía exhaustiva con checklists

### 4. Trámites del SAT

Información detallada sobre trámites:

- Alta en RFC
- Renovación e.firma
- Actualización de datos
- Cambio de régimen
- Suspensión de actividades
- Y más...

**Incluye:**
- Requisitos
- Documentación necesaria
- Pasos detallados
- Tiempos de respuesta
- Costos
- Enlaces oficiales

### 5. Ayuda con Declaraciones

Asistencia para preparar y presentar declaraciones:

- Declaraciones mensuales
- Declaraciones anuales
- Información necesaria
- Cómo llenar formatos
- Validaciones
- Errores comunes
- Pasos de presentación

### 6. Devoluciones y Compensaciones

Asesoría sobre:
- Devoluciones de saldo a favor
- Compensaciones
- Cuándo aplicar cada opción
- Requisitos y tiempos
- Seguimiento

## 📊 Regímenes Fiscales Soportados

1. **RESICO** - Régimen Simplificado de Confianza
2. **PFAE** - Personas Físicas con Actividades Empresariales
3. **Plataformas** - Régimen para plataformas digitales
4. **Sueldos y Salarios** - Asalariados
5. **Personas Físicas** - Otros regímenes
6. **Personas Morales** - Empresas

## 🔧 Tecnologías

- **OpenRouter**: Acceso a modelos de IA avanzados
- **FastAPI**: API REST moderna y rápida
- **Pydantic**: Validación de datos
- **httpx**: Cliente HTTP asíncrono
- **Python 3.8+**: Lenguaje base
