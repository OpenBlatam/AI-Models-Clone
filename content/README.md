## 📋 Overview

The **Content** module serves as the authoritative source for written materials within the Blatam Academy. It utilizes the power of **MDX** (Markdown with JSX) to enable interactive documentation and rich-media blog posts. This directory is typically processed by Contentlayer to provide type-safe data to the Next.js frontend.

## 📁 Structure

```
content/
├── blog/                 # Public-facing articles and marketing posts
├── docs/                 # Technical documentation and API references
├── guides/               # Tutorial-style learning paths and onboarding
└── pages/                # Static site pages (About, Terms, etc.)
```

## 🚀 Key Features

| Feature | Description |
|---------|-------------|
| **MDX Support** | Embed React components directly within markdown files. |
| **Type-Safe** | Integrated with Contentlayer for build-time validation. |
| **SEO Ready** | Organized structure for optimal search engine indexing. |
| **Multi-Category** | Logical separation between learning materials and marketing content. |
