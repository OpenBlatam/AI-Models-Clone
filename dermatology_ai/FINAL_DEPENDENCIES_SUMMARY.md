## 🚀 Comandos Principales

### Instalación
```bash
make install          # Producción completa
make install-opt      # Optimizado (recomendado)
make install-dev      # Desarrollo
make install-gpu      # Con GPU
```

### Gestión
```bash
make update           # Actualizar
make check            # Verificar seguridad
make outdated         # Ver desactualizadas
make backup           # Backup
make restore          # Restore
```

### Análisis
```bash
make check-all        # Todos los checks
make validate-reqs    # Validar formato
make dep-tree         # Árbol de dependencias
make dashboard        # Dashboard interactivo
```

### Desarrollo
```bash
make test             # Tests
make lint             # Linters
make format           # Formatear
make pre-commit       # Pre-commit
```
