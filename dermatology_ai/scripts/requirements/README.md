# Requirements Management Scripts

Sistema organizado de scripts para gestión de dependencias.

## 📁 Estructura

- **analysis/** - Scripts de análisis y visualización
- **validation/** - Scripts de validación y verificación
- **management/** - Scripts de gestión y mantenimiento
- **utils/** - Utilidades y herramientas auxiliares

## 🚀 Uso Rápido

```bash
# Usar Makefile (recomendado)
make help              # Ver todos los comandos
make check             # Verificar seguridad
make analyze           # Análisis completo

# O usar scripts directamente
./scripts/requirements/analysis/analyze-dependencies.py
./scripts/requirements/validation/check-dependencies.sh
```
