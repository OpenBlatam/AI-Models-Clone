## 📝 Ejemplos de Uso por Categoría (Actualizado Comprehensivo)

### Configuration Testing
```python
from tests.test_config_helpers import (
    create_mock_config,
    with_env_vars,
    assert_config_value
)

with with_env_vars({"DATABASE_URL": "sqlite:///test.db"}):
    config = create_mock_config({"database_url": os.getenv("DATABASE_URL")})
    assert_config_value(config, "database_url", "sqlite:///test.db")
```

### Retry Testing
```python
from tests.test_retry_helpers import (
    test_retry_mechanism,
    assert_retry_occurred
)

result = await test_retry_mechanism(
    failing_function,
    max_retries=3,
    retry_delay=0.1
)
assert_retry_occurred(result, expected_attempts=3)
```

### Observability Testing
```python
from tests.test_observability_helpers import (
    create_mock_observability_system,
    assert_trace_recorded
)

system = create_mock_observability_system()

```
