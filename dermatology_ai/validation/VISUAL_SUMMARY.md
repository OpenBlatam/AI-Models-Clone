## 🎯 Criterios de Decisión

```
┌─────────────────────────────────────────────────────────────┐
│                    CRITERIOS DE DECISIÓN                     │
└─────────────────────────────────────────────────────────────┘

✅ CONTINUAR SI:
   ├─ Tasa conversión > 20%
   ├─ Satisfacción > 3.5/5
   ├─ NPS > 30
   ├─ Intención pago > 30%
   └─ 100+ signups (si aplica)

⚠️ ITERAR SI:
   ├─ Tasa conversión 10-20%
   ├─ Satisfacción 3.0-3.5/5
   ├─ Feedback mixto pero hay interés
   └─ Problemas técnicos solucionables

❌ PIVOTAR SI:
   ├─ Tasa conversión < 10%
   ├─ Satisfacción < 3.0/5
   ├─ NPS < 0
   └─ No hay interés después de 2 semanas
```
