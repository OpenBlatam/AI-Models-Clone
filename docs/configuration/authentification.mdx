# Configuración de Autenticación

## Proveedores de Autenticación

<h4 className="font-semibold text-gray-800">1. Configuración de Google OAuth</h4>
<p>
  Para configurar la autenticación con Google, necesitamos crear un proyecto en Google Cloud Console y obtener las credenciales OAuth necesarias.
</p>

<h4 className="font-semibold text-gray-800">2. Configuración de GitHub OAuth</h4>
<p>
  La autenticación con GitHub requiere la creación de una aplicación OAuth en GitHub y la configuración de las credenciales correspondientes.
</p>

## Base de Datos

<h4 className="font-semibold text-gray-800">3. Configuración de Prisma</h4>
<p>
  Prisma se utiliza como adaptador de base de datos para Next Auth. Necesitamos configurar el esquema y las migraciones correspondientes.
</p>
