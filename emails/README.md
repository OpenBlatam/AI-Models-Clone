## 📋 Overview

The **Emails** module contains high-quality React-based email templates. By leveraging frameworks like **React Email**, these templates ensure consistent rendering across all major email clients while allowing developers to use standard React patterns for dynamic content and styling.

## 🚀 Key Templates

| Template | Description |
|---------|-------------|
| `magic-link-email.tsx` | Secure login notification with passwordless authentication links. |

## 💻 Tech Stack

- **React Email**: Core template engine.
- **Tailwind CSS**: For unified, inline-compatible styling.
- **TypeScript**: Ensuring prop safety for dynamic variables.
