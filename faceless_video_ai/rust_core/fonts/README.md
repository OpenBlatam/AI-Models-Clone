# Fonts Directory

Este directorio contiene las fuentes utilizadas por el módulo de procesamiento de imágenes.

## Fuentes Requeridas

Para el funcionamiento del watermarking de texto, necesitas agregar:

- `DejaVuSans.ttf` - Fuente principal para texto

## Cómo Obtener las Fuentes

### Opción 1: Descargar DejaVu Fonts
```bash
# Linux/Ubuntu
sudo apt-get install fonts-dejavu

# O descarga desde:
# https://dejavu-fonts.github.io/Download.html
```

### Opción 2: Copiar desde Sistema
```bash
# Linux
cp /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf ./

# Windows
copy C:\Windows\Fonts\DejaVuSans.ttf ./

# macOS
cp /System/Library/Fonts/Supplemental/DejaVuSans.ttf ./
```

### Opción 3: Usar Fuentes Alternativas
Cualquier fuente TTF puede funcionar, solo renómbrala a `DejaVuSans.ttf` o modifica el código fuente.
