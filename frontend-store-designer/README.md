# Physical Store Designer AI - Frontend

Frontend Next.js 15.3 con TypeScript y Turbopack para el sistema de diseño de locales físicos con IA.

## Características

- Next.js 15.3 con App Router
- TypeScript
- Turbopack para desarrollo rápido
- React Query para manejo de estado del servidor
- Tailwind CSS para estilos
- Componentes UI modernos
- Integración completa con la API del backend

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Configuración

Crea un archivo `.env.local` con:

```
NEXT_PUBLIC_API_URL=http://localhost:8030
```

## Estructura

- `src/app/` - Páginas y rutas
- `src/components/` - Componentes reutilizables
- `src/lib/` - Utilidades y cliente API
- `src/types/` - Tipos TypeScript

## Funcionalidades

- Chat interactivo con IA
- Generación de diseños de locales
- Visualización de diseños
- Análisis y reportes
- Dashboard con métricas
