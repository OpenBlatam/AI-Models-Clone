## 📋 Overview

The **Guides** module offers deep-dive instructional content. While the `content/docs` folder focuses on reference material, this module is dedicated to tutorials, learning paths, and best practices. These guides are written in **MDX** to support interactive code snippets and embedded visual aids.

## 🚀 Available Guides

| Guide | Description |
|---------|-------------|
| `installation.mdx` | Step-by-step setup guide for the local development environment. |
| `learning-path.mdx` | Structured curriculum for mastering the platform's AI features. |
| `using-next-auth...` | Deep dive into the authentication flow with NextAuth. |
