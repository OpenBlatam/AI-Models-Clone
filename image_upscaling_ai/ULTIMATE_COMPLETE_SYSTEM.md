## 🔧 Ejemplos de Uso

### Performance

```python
# Perfilar operación
profile = upscaler.profile_upscale("image.jpg", 2.0, "real_esrgan_like")

# Detectar cuellos de botella
bottlenecks = upscaler.get_performance_bottlenecks(profile)

# Optimizar rendimiento
optimization = upscaler.optimize_performance("image.jpg", 2.0, target_time=5.0)

# Uso de recursos
resources = upscaler.get_resource_usage()
print(f"CPU: {resources['cpu_percent']}%")
print(f"Memory: {resources['memory_rss_mb']} MB")
```

### Workflow

```python
# Crear workflow personalizado
upscaler.create_workflow(
    "my_workflow",
    [
        {"type": "upscale", "params": {"method": "real_esrgan_like"}},
        {"type": "enhance", "params": {"enhancement": "edges", "strength": 1.2}},
        {"type": "validate", "params": {}},
    ]
)

# Ejecutar workflow
result = upscaler.execute_workflow("my_workflow", "image.jpg", 2.0)

# Crear workflow estándar
workflow_name = upscaler.create_standard_workflow("quality")
result = upscaler.execute_workflow(workflow_name, "image.jpg", 2.0)

# Listar workflows
workflows = upscaler.list_workflows()
```
