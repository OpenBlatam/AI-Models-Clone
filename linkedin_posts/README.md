## 💻 Basic Usage

```python
from linkedin_posts.main import LinkedInPostsGenerator

# Initialize generator
generator = LinkedInPostsGenerator()

# Generate post
post = generator.generate(
    topic="AI Trends",
    audience="professionals",
    tone="professional"
)
```
