# 🚀 Start Here - Test Suite

## 👋 Bienvenido a la Suite de Tests

Esta es la suite de tests completa del frontend de Music Analyzer AI.

## ⚡ Inicio Rápido

```bash

npm test

# Con cobertura
npm run test:coverage

# Modo watch
npm run test:watch
```
