# Constants

This directory contains all application constants.

## Usage

```typescript
import { 
  API_ENDPOINTS, 
  QUERY_KEYS, 
  ROUTES, 
  STORAGE_KEYS,
  PAGINATION,
  DEBOUNCE_DELAYS,
  TIMEOUTS,
  VALIDATION_LIMITS,
  FEATURES 
} from '@/lib/constants';
```

## Categories

- **API_ENDPOINTS**: API endpoint paths
- **QUERY_KEYS**: React Query cache keys
- **ROUTES**: Application routes
- **STORAGE_KEYS**: LocalStorage keys
- **PAGINATION**: Pagination defaults
- **DEBOUNCE_DELAYS**: Debounce timing
- **TIMEOUTS**: Timeout values
- **VALIDATION_LIMITS**: Validation constraints
- **FEATURES**: Feature flags
