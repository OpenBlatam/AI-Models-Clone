## 📋 Overview

The **Pages** module contains entry points for the Next.js **Pages Router**. As the platform transitions to the App Router (`/app`), this directory maintains core legacy pages, global application overrides (`_app.tsx`), and supplementary routes that require independent lifecycle management.

## 📁 Key Files

| File | Description |
|---------|-------------|
| `_app.tsx` | Global application wrapper, handling shared state and styles. |
