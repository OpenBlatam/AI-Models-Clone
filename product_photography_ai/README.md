# Product Photography AI

> Part of the [Blatam Academy Integrated Platform](../README.md)

## 📋 Description

AI system for generating professional product photography.

## 🚀 Features

- **Background Removal**: Automatic background removal
- **Scene Generation**: Generate realistic scenes for products
- **Lighting Adjustment**: AI-powered lighting enhancement

## 📁 Structure

```
product_photography_ai/
├── processor.py           # Image processing
├── generator.py           # Scene generation
└── models/                # AI models
```

## 🔧 Installation

```bash
pip install -r requirements.txt
```
