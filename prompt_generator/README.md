# Prompt Generator

> Part of the [Blatam Academy Integrated Platform](../README.md)

## 📋 Description

Tool for generating and optimizing prompts for various AI models.

## 🚀 Features

- **Prompt Optimization**: Refines prompts for better results
- **Template Library**: Collection of proven prompt templates
- **Multi-Model Support**: Adapted prompts for different models (GPT, Claude, etc.)

## 📁 Structure

```
prompt_generator/
├── optimizer.py           # Optimization logic
├── templates/             # Prompt templates
└── models/                # Model-specific adapters
```

## 🔧 Installation

```bash
pip install -r requirements.txt
```
