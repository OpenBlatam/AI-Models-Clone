# Sales Agent

> Part of the [Blatam Academy Integrated Platform](../README.md)

## 📋 Description

Autonomous agent for managing sales processes and customer interactions.

## 🚀 Features

- **Lead Qualification**: Automatically qualify leads
- **Follow-Up Automation**: Schedule and send follow-up messages
- **Deal Closing**: Assist in closing sales deals

## 📁 Structure

```
sales_agent/
├── agent.py               # Sales logic
├── pipelines.py           # Sales pipelines
└── crm.py                 # CRM integration
```

## 🔧 Installation

```bash
pip install -r requirements.txt
```
