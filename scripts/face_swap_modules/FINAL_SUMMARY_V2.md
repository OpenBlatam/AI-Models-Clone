## 🚀 Uso de Nuevas Herramientas

### Benchmark de Rendimiento

```bash
# Benchmark básico
python face_swap_modules/benchmark.py

# Con imagen específica
python face_swap_modules/benchmark.py -i test_image.jpg

# Más iteraciones
python face_swap_modules/benchmark.py -n 20

# Guardar resultados
python face_swap_modules/benchmark.py -o results.txt
```

### Demostración Visual

```bash
# Demostración completa
python face_swap_modules/demo.py source.jpg target.jpg

# Directorio de salida personalizado
python face_swap_modules/demo.py source.jpg target.jpg -o output_dir
```

### Tests

```bash
# Tests unitarios
python -m pytest tests/test_base.py

# Tests de integración
python -m pytest tests/test_integration.py

# Todos los tests
python -m pytest tests/
```
