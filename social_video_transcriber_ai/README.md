# Social Video Transcriber AI

> Part of the [Blatam Academy Integrated Platform](../README.md)

## 📋 Description

Tool for transcribing videos from social networks.

## 🚀 Features

- **Link Support**: Transcribe from URLs (YouTube, TikTok, Instagram)
- **Automatic Language Detection**: Detects and transcribes multiple languages
- **SRT Export**: Export subtitles in standard formats

## 📁 Structure

```
social_video_transcriber_ai/
├── downloader.py          # Video downloader
├── transcriber.py         # Transcription engine
└── formatter.py           # Subtitle formatter
```

## 🔧 Installation

```bash
pip install -r requirements.txt
```
